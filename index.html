SismoAlerta

Alerta sísmica inmediata para instituciones educativas.

Conocer más

¿Qué problema resuelve?

Las instituciones educativas requieren una forma rápida y confiable de alertar a los estudiantes y personal ante movimientos sísmicos. SismoAlerta permite detectar sismos en tiempo real y emitir avisos inmediatos para una evacuación segura.

¿Para quién está diseñado?

Administradores

Supervisan el estado del sistema y gestionan configuraciones.

Técnicos

Monitorean nodos y mantienen la red de sensores operativa.

Docentes

Utilizan el modo simulacro para entrenar a estudiantes.

Características principales

  • ✅ Monitoreo en tiempo real de nodos sísmicos.
  • ✅ Alertas visuales y auditivas inmediatas.
  • ✅ Registro de eventos para análisis posterior.
  • ✅ Gestión de usuarios y roles de acceso.
  • ✅ Modo de simulacro para prácticas educativas.
  • ✅ Exportación de datos para informes.

Producto Mínimo Viable (PMV)

La versión inicial demuestra el funcionamiento básico del sistema:

  • • Monitoreo de estado de nodos (activo/inactivo).
  • • Señales visuales de alerta ante desconexión o reconexión.
  • • Registro simple de eventos para auditoría.
